# 区域活动参与度分析模型

## 项目简介

本项目是一个区域活动参与度数据分析项目，通过分析活动区域的客流量、用户画像、驻留时长等多维度数据，计算活动参与度评分，为区域活动效果评估提供数据支持。

## 数据说明

### 数据来源
- **无锡映月湖**: 包含九里仓、映月天地、映月湖中央公园、映月湖公园等子区域
- **江阴音乐节**: 包含江阴体育馆、次级防控圈等活动区域

### 数据字段
| 字段名 | 说明 |
|--------|------|
| area | 活动区域 |
| date | 日期 |
| time | 时段 |
| p_flow | 客流量 |
| age_1/age_2/age_3 | 年龄段分布（0-25/26-44/45-60+） |
| first_visit | 首次到访占比 |
| man/woman | 男女客流量 |
| dur_1/dur_2/dur_3/dur_4 | 驻留时长分布 |
| label | 是否为活动时间（0/1） |

## 分析方法

### 参与度评分模型

本项目采用加权评分模型计算活动参与度，主要考虑以下维度：

1. **客流量变化** (权重: 37%)
   - 活动当天客流量 vs 前七天平均客流量
   - 增长率贡献: 5%

2. **首次到访占比** (权重: 27%)
   - 衡量活动对新用户的吸引力

3. **驻留时长** (权重: 14%)
   - 不同时长区间的人数分布

4. **用户均衡度** (权重: 5%)
   - 性别比例均衡
   - 年龄段分布均衡

### 数据预处理
- 使用 MinMaxScaler 进行数据归一化
- 计算活动当天与非活动时间的对比指标

## 使用方法

### 环境要求
```bash
pip install pandas numpy scikit-learn
```

### 运行脚本
```bash
python aup_analysis.py
```

### 输出结果
脚本将输出活动当天各时段的参与度得分，格式如下：
```
活动当天各时段的参与度得分:
       area      date  time  score
0  无锡映月湖  20240501     0  0.xxx
1  无锡映月湖  20240501     1  0.xxx
...
```
