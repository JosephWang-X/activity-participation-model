📊 区域活动参与度分析模型

基于多维度数据的活动效果评估系统

🎯 项目简介

本项目是一个区域活动参与度数据分析项目，通过分析活动区域的客流量、用户画像、驻留时长等多维度数据，计算活动参与度评分，为区域活动效果评估提供数据支持。

📈 客流量分析

对比活动当天与历史数据，评估活动吸引力

👥 用户画像

分析年龄、性别分布，评估用户均衡度

⏱️ 驻留时长

统计不同停留时长，衡量活动粘性

🆕 新用户吸引

首次到访占比反映活动拉新效果

📊 数据说明

数据来源

  • 无锡映月湖: 包含九里仓、映月天地、映月湖中央公园、映月湖公园等子区域
  • 江阴音乐节: 包含江阴体育馆、次级防控圈等活动区域

数据字段

字段名 说明
area活动区域
date日期
time时段
p_flow客流量
age_1/age_2/age_3年龄段分布（0-25/26-44/45-60+）
first_visit首次到访占比
man/woman男女客流量
dur_1/dur_2/dur_3/dur_4驻留时长分布
label是否为活动时间（0/1）

🔧 使用方法

环境要求

pip install pandas numpy scikit-learn

运行脚本

python aup_analysis.py

输出结果

脚本将输出活动当天各时段的参与度得分
